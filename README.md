# Multi-modal-Accessibility-Framework-for-the-Visually-Impaired

This project presents an intelligent assistive system designed to support visually impaired individuals by summarizing lengthy text documents and converting them into **Braille** and **speech outputs**. It compares both **abstractive** and **extractive** summarization models and evaluates how effectively the generated summaries retain meaning after Braille translation.

---

# Key Features

-  **Text Summarization**
  - Abstractive Models: **T5, PEGASUS, BART, BERT**
  - Extractive Models: **TextRank, LSA, Luhn**
-  **Braille Conversion**
  - Converts summaries into Unicode Braille patterns
  - Supports refreshable Braille display & printing
-  **Text-to-Speech (TTS)**
  - Uses Google Text-to-Speech (gTTS) for audio output
- **Bidirectional Conversion**
  - **Braille ‚ûù English** and **English ‚ûù Braille**
- **Model Evaluation**
  - ROUGE, BLEU, Levenshtein distance metrics
